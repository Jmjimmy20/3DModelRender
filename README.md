# WebXR Learning Labs

A collection of practical experiments focused on learning WebXR technologies, Three.js, WebVR, WebAR, and interactive 3D experiences for the web.

The goal of this repository series is to document my learning journey while building small prototypes and experiments that progressively explore the capabilities of modern web-based immersive technologies.

---

# Lab 01 - 3D Renderind with Three.js

## Overview

This first experiment introduces the fundamentals of 3D rendering on the web using Three.js.

The objective of this lab is to understand how to create and manage a 3D scene, render primitive geometries, load external 3D assets, and interact with objects through camera controls.

## What I Learned

- Setting up a basic Three.js project
- Creating and rendering primitive 3D geometries
- Loading external 3D models into a scene
- Orbiting around objects
- Zooming in and out using camera controls
- Managing lights, camera, and scene composition
- Understanding the basics of the Three.js rendering pipeline

## Features

- Primitive 3D objects rendered with Three.js
- External 3D model integration
- Object visualization and inspection
- Interactive camera controls
  - Rotation
  - Pan
  - Zoom

## Purpose

This lab serves as the foundation for future experiments involving:

- WebXR
- Virtual Reality (VR)
- Augmented Reality (AR)
- Interactive 3D environments
- Real-time rendering techniques
